# Krassembly
A simple assembly like language written in C. 

This language will support macros, labels, local labels, parameters (labels as functions),
variables and local variables.

In the future i want to add more features like enums, structures and type checking macros.

## Examples
You can look in to the examples dir for some examples.

## Intentions
I wanted to learn a bit more C and write some C so i created this language.
